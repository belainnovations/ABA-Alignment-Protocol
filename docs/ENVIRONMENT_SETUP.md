# Environment Setup & Configuration

## Active Python Interpreter
**Crucial for Agent Execution:** All commands must utilize this absolute path to access the correct Conda environment.

*   **Environment Name:** `aba_protocol_env`
*   **Python Executable Path:** `C:\Users\User\anaconda3\envs\aba_protocol_env\python.exe`

## Usage Usage
Instead of running `python script.py`, use:
```powershell
C:\Users\User\anaconda3\envs\aba_protocol_env\python.exe script.py
```

Instead of running `pip install ...`, use:
```powershell
C:\Users\User\anaconda3\envs\aba_protocol_env\python.exe -m pip install ...
```
